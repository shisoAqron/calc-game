# プロジェクト概要

小学生が計算練習をするためのアプリ

## アプリケーション仕様・要望

- スマホからアクセスするので，レイアウトはシンプルにしたい
  - ネイティブのキーボードで入力させるよりも，画面にボタンを置いてそれをタップするのがよさそう
  - 案としては2つ
    - キーボードを模したボタンをおいて答えを入力させる
    - 答えを3択として出して選ばせる
- ゲームっぽさを出したい
  - 時間制限を設ける: 爆弾と導火線で爆発する前に答えるとか
  - 正解・不正解演出: 上の例だと不正解なら爆発で派手さはある．正解の場合も楽しさが出るようにしたい
    - ボタンの上にエフェクトをオーバーレイすることで，派手さとボタンの押しやすさを両立する
  - アニメーションはできる限り採用．後述の公開方法でできる方法で実現する
- github pagesで公開したい
  - メンテを少なくしたいのでサーバレスで動かしたい．
  - ページ遷移としては次を想定
    - トップページ: 問題種別もここで選ぶ
    - ゲームページ: トップに戻るボタンやリザルトを確認した後はトップに戻る
- 一時的な生成ファイルは作業時に削除してください
